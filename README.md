# Medieval Overhaul - No Schematics

Removes the schematic requirement from Medieval Overhaul research projects and removes the ~16 research schematic scrolls from the game entirely.

Standalone patch mod — does not modify Medieval Overhaul's own files. Load it AFTER Medieval Overhaul (and after any of its sub-patches such as Biotech). All changes are XML PatchOperations; safe to add to or remove from an existing save.

Researches affected become freely researchable as soon as their normal prerequisites are met. The lore/quest scrolls (The Beginning, Into the Ruins, Ancient Codex) are left untouched.
